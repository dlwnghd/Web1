JAVA - 프로그래밍 언어
	프로그래밍 언어는 개발자와 운영체제와 소통하기 위한 언어이다.

프로그램
	소스코드로 잘 짜여진 틀.

	프로그램
	OS(Operating System : 운영체제)
	하드웨어

	- 일반 프로그램은 이식성이 좋지 않다.

	JAVA 프로그램
	JVM	: JAVA 프로그램은 이식성이 좋다.

소스코드
	명령어를 작성해 놓은 것.
	운영체제와 개발자가 소통할 것을 글로 작성해 놓은 것.
콘솔
	사용자와 운영체제가 소통한 결과를 나타내는 창.

컴파일
	사람의 언어를 컴퓨터 언어로 바꾸어주는 작업 or 번역기.
	JAVA -> 컴퓨터 언어
컴파일러
	컴파일을 해주는 프로그램 또는 명령어.
------------------------------------------------------------------------
JVM(Java Virtual Machine)			꽃
	JAVA 프로그램을 실행해 줌.

JRE(Java Runtime Environment)		환경
	JVM을 생성하는 부분.
	JVM을 실행할 때 필요한 라이브러리 파일들을 가지고 있다.

JDK(Java Development Kit)			물
	JRE 외에 개발에 필요한 도구들을 가지고 있다.
	컴파일 명령어와 실행 명령어를 담고 있다.
